# templates/word

本目录存放 **协同批量生成Word.html** 可使用的 Word 模板（.docx）。

## 命名建议
- 用中文或英文描述用途，例如 `派工单模板.docx`、`现场签证单.docx`。
- 文件名会直接显示在使用者列表里，起个好懂的名字。

## 占位符规范
模板里用 `{{数据001}}`、`{{数据002}}` …… 表示要替换的字段，
**必须写成一整段不被拆分的文字**（不要一半加粗一半不加粗、不要换行），否则网页无法识别。

## 上传方式
在 `协同批量生成Word.html` 的「共享模板库」卡片中：填好仓库/分支/目录（本仓库已预填），
保存有 `repo` 权限的 Token，然后选本地 .docx 点「上传到库」即可。
