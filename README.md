# 协同批量生成 Word · 共享模板库

本仓库配合单文件离线工具 **`协同批量生成Word.html`** 使用，用于**多人共享 Word 模板**：

- 任何人无需登录即可 **浏览 / 下载** 本仓库 `templates/word/` 下的模板；
- 上传模板需要 GitHub Token（有写权限），仅保存在上传者本机浏览器。

## 快速开始（使用者）

1. 下载本仓库里的 **`协同批量生成Word.html`**（单文件，双击即可在浏览器打开，无需联网安装）。
2. 准备一份 Excel 数据表（表头用 `数据001`、`数据002`…… 表示字段，例如 `数据001`=合同名称、`数据002`=编号）。
3. 在网页「共享模板库（GitHub）」卡片已预填好本仓库，直接点 **连接 / 刷新列表**，即可看到模板，点 **使用** 或 **下载**。
4. 上传你的 Excel + 选中的模板，点生成，即可得到批量 Word 文档。

## 模板占位符规范（制作者必读）

在 Word 模板里，用 `{{数据001}}`、`{{数据002}}` …… 表示要被替换的字段：

- 占位符必须写成 **一整段不被拆分的文字**（不要「数据」加粗、「001」不加粗；不要中间换行）。
- 被拆分会导致网页检测不到 / 替换失败。最简单做法：删掉原来的，重新纯文本手打一遍 `{{数据001}}`。
- 网页仅认 `{{数据NNN}}` 形式（NNN 为 001~999）。

## 如何新增模板（上传者）

- 在网页里点「上传到库」选择本地 `.docx`，或「上传当前模板到库」。
- 先在「Token」框填入你的 GitHub Personal Access Token（需 `repo` 权限），点 **保存 Token**。

## 仓库结构

```
word-templates/
├─ README.md                  # 本说明
├─ 协同批量生成Word.html       # 工具本体（单文件离线）

   └─ word/

      └─ 模板.docx             # 示例种子模板（派工单/现场签证单）
```
